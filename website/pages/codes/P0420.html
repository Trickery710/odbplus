P0420

Catalyst System Efficiency Below Threshold (Bank 1)

Powertrain Generic (SAE) Severity: Medium

System

Emissions / Exhaust

Subsystem

Catalytic Converter

Common Fix

Replace Catalytic Converter

Repair Cost

$100 - $2,500

What Does P0420 Mean?

The P0420 code indicates that the catalytic converter on Bank 1 (the side with cylinder #1) is not working efficiently enough to meet emissions standards. The Engine Control Module (ECM) monitors catalyst efficiency by comparing the signals from the upstream (pre-cat) and downstream (post-cat) oxygen sensors.

A healthy catalytic converter stores and releases oxygen as exhaust gases pass through it. This causes the downstream O2 sensor to have a relatively steady voltage, while the upstream sensor switches rapidly between rich and lean. When the downstream sensor starts switching similarly to the upstream sensor, it indicates the catalyst isn't storing oxygen properly - it's "worn out."

Don't Immediately Replace the Cat: While a failed catalytic converter is the most common cause of P0420, other issues like exhaust leaks, faulty O2 sensors, or engine problems can trigger this code. Always diagnose before replacing expensive parts.

How the Catalytic Converter Works

The catalytic converter is an emissions control device that converts harmful pollutants in exhaust gas into less harmful substances through chemical reactions:

  • Carbon Monoxide (CO) → Carbon Dioxide (CO2)
  • Hydrocarbons (HC) → Water (H2O) + Carbon Dioxide (CO2)
  • Nitrogen Oxides (NOx) → Nitrogen (N2) + Oxygen (O2)

Catalyst Efficiency Monitoring

The ECM monitors catalyst efficiency by watching the downstream O2 sensor:

  • Good catalyst: Downstream O2 sensor voltage is relatively flat/stable (0.5-0.7V)
  • Failing catalyst: Downstream sensor switches similarly to upstream sensor
  • Monitor conditions: Runs after engine is warm, steady cruise, no other faults present

The ECM compares the "switching ratio" of both sensors. When the downstream sensor switches too frequently (indicating the cat isn't buffering oxygen properly), P0420 is set.

Common Symptoms

  • Check Engine Light

    Usually steady (not flashing) check engine light

  • Failed Emissions Test

    Vehicle will fail emissions/smog testing with this code

  • Often No Driveability Issues

    Many drivers notice no change in how the vehicle runs

  • Slight Power Loss (Severe Cases)

    If the catalyst is clogged, you may notice reduced power

  • Rotten Egg Smell (Sometimes)

    Sulfur smell may indicate a failing or overwhelmed catalyst

Possible Causes

Catalytic Converter Issues

  • Worn/degraded catalyst (most common)
  • Catalyst contaminated by oil or coolant
  • Catalyst damaged by overheating
  • Physical damage to catalyst substrate

O2 Sensor Problems

  • Faulty downstream O2 sensor
  • Lazy/slow responding O2 sensor
  • O2 sensor wiring issues
  • Incorrect O2 sensor installed

Exhaust Leaks

  • Exhaust leak before downstream sensor
  • Cracked exhaust manifold
  • Leaking exhaust gaskets
  • Holes in exhaust pipes

Engine Issues

  • Misfires sending unburned fuel to cat
  • Rich running condition
  • Oil burning (contaminating cat)
  • Coolant leak into combustion
Fix Root Causes First: If your engine has been running rich, misfiring, or burning oil, fix those issues BEFORE replacing the catalytic converter. Otherwise, the new cat will be damaged by the same problems.

Diagnostic Steps

1

Check for Other Codes

Before diagnosing P0420, check for and repair any other codes, especially misfires (P0300-P0308), fuel system codes (P0171/P0172), or O2 sensor codes. These issues can cause false P0420 codes or damage the catalyst.

2

Monitor O2 Sensor Data

Use your ODB+ scanner to graph both O2 sensor signals simultaneously. The upstream sensor should switch rapidly (crossing 0.45V multiple times per second). The downstream sensor should be relatively stable. If downstream mirrors upstream, the catalyst is likely bad.

3

Check Exhaust for Leaks

Inspect the exhaust system from the manifold to beyond the downstream O2 sensor. Look for rust holes, loose connections, or damaged gaskets. An exhaust leak can introduce oxygen that affects the downstream sensor reading.

4

Test O2 Sensor Response

With engine warm and running, create a lean condition (disconnect vacuum hose) - upstream sensor should go low quickly. Then create rich condition (propane enrichment) - sensor should go high quickly. A slow-responding sensor can cause false P0420.

5

Check Catalyst Temperature

Using an infrared thermometer, check the temperature at the inlet and outlet of the catalytic converter during steady operation. The outlet should be hotter than the inlet (50-100°F higher) if the cat is working. Similar temps or cooler outlet may indicate a dead catalyst.

Common Repairs

Replace Downstream O2 Sensor

Try this first if the sensor is old or responding slowly.

DIY: $30-$100 Shop: $100-$200

Repair Exhaust Leak

Fix any leaks between the engine and downstream sensor.

DIY: $20-$50 Shop: $100-$300

Replace Catalytic Converter (Aftermarket)

Aftermarket catalysts are less expensive but may not last as long.

DIY: $150-$400 Shop: $400-$800

Replace Catalytic Converter (OEM)

OEM catalysts are more expensive but typically last longer and work better.

DIY: $500-$1,500 Shop: $1,000-$2,500
California & CARB States: In California and other states that follow CARB regulations, you must use a CARB-compliant catalytic converter. These are more expensive but required by law.
