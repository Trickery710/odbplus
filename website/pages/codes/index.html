Understanding OBD-II Code Format

P 0 3 00
P System: P=Powertrain, B=Body, C=Chassis, U=Network
0 Type: 0=Generic (SAE), 1=Manufacturer-specific
3 Subsystem: 1=Fuel/Air, 2=Fuel/Air, 3=Ignition, 4=Emissions, etc.
00 Specific Fault: Identifies the exact problem within the subsystem
