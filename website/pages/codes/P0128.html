P0128

Coolant Thermostat (Coolant Temperature Below Thermostat Regulating Temperature)

Cooling System Generic (SAE) Severity: Low-Medium

System

Engine Cooling

Subsystem

Thermostat Control

Common Fix

Replace Thermostat

Repair Cost

$75 - $250

What Does P0128 Mean?

The P0128 code indicates that the engine is not reaching its expected operating temperature within a specified time period. The powertrain control module (PCM) monitors coolant temperature during warm-up and expects it to reach a certain threshold (typically around 160-180째F) within a certain time based on ambient conditions.

When the engine stays too cool, the PCM sets this code. The most common cause is a stuck-open thermostat, which allows coolant to flow to the radiator constantly instead of letting the engine warm up properly.

Why It Matters: A cool-running engine burns more fuel, produces more emissions, and experiences more wear. The engine is designed to operate at a specific temperature range for optimal efficiency and longevity.

How the Cooling System Works

The cooling system maintains optimal engine temperature through a carefully controlled balance of coolant flow:

Cooling System Components:

  • Thermostat: A temperature-controlled valve that regulates coolant flow. Stays closed when cold, opens when engine reaches operating temperature.
  • Water Pump: Circulates coolant through the engine and radiator
  • Radiator: Transfers heat from the coolant to the outside air
  • Coolant Temperature Sensor (ECT): Reports coolant temperature to the PCM
  • Cooling Fan: Provides additional airflow through radiator when needed

Normal Operation: When you start a cold engine, the thermostat is closed. Coolant circulates only through the engine block, allowing it to warm up quickly. Once the coolant reaches the thermostat's rated temperature (typically 195-205째F), the thermostat opens and allows coolant to flow through the radiator.

Common Symptoms

  • Temperature Gauge Never Reaches Normal

    Gauge stays below the normal operating range, especially in cold weather

  • Poor Heater Performance

    Cabin heater blows lukewarm air even after extended driving

  • Decreased Fuel Economy

    Cold engine requires richer fuel mixture, wasting fuel

  • Check Engine Light

    P0128 code stored, may take several drive cycles to set

  • Extended Warm-up Time

    Takes much longer than normal for engine to warm up

  • Higher Emissions

    Cold engines produce more pollutants, may fail emissions test

Possible Causes

Thermostat Issues (Most Common)

  • Thermostat stuck open
  • Thermostat opening too early
  • Thermostat missing (removed)
  • Wrong thermostat rating installed
  • Thermostat installed incorrectly

Coolant Temperature Sensor

  • Faulty ECT sensor reading low
  • Poor sensor connection
  • Corroded sensor terminals
  • Wrong sensor for application

Cooling System Problems

  • Low coolant level
  • Air pockets in cooling system
  • Cooling fan running constantly
  • Radiator hose collapsed internally

Other Causes

  • PCM software issue
  • Extreme cold weather operation
  • Wiring problem to ECT sensor
  • Engine running too lean (rare)

Diagnostic Steps

1

Check Coolant Level and Condition

Verify coolant is at the proper level when cold. Low coolant can affect temperature readings. Check for contamination (oil, rust) that could affect cooling system operation.

2

Monitor ECT with Scanner

Start with a cold engine and use your ODB+ scanner to monitor coolant temperature. It should start at approximately ambient temperature (matching IAT if cold soaked) and rise steadily to 195-220째F. If it never reaches normal or rises very slowly, suspect the thermostat.

3

Feel the Radiator Hoses

During warm-up, feel the upper radiator hose. It should remain cool/cold until the thermostat opens (around 195째F). If it gets hot immediately, the thermostat is stuck open. Caution: components get hot!

4

Check Cooling Fan Operation

The cooling fan should not run during warm-up on a cold day unless A/C is on. A constantly running fan will prevent the engine from reaching operating temperature. Check fan relay and temperature switch.

5

Test the ECT Sensor

Compare scanner ECT reading to actual coolant temperature using an infrared thermometer on the thermostat housing. If readings differ significantly, the ECT sensor may be faulty.

Common Repairs

Replace Thermostat

Most common fix. Use OEM or quality aftermarket thermostat with correct temperature rating.

DIY: $20-$50 Shop: $150-$300

Replace ECT Sensor

If sensor is reading inaccurately, replacement is straightforward.

DIY: $15-$40 Shop: $75-$150

Repair Wiring/Connector

Fix corroded or damaged wiring to ECT sensor.

DIY: $5-$20 Shop: $50-$150

Cooling System Flush and Fill

May be needed along with thermostat replacement.

DIY: $30-$50 Shop: $100-$200
