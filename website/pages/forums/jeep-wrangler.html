Common Issues by Generation

JL/JLU (2018-Present)

  • Death wobble - Front end shimmy at highway speeds
  • Welded frame recall - Frame weld quality issues
  • Clutch issues - Manual trans clutch problems
  • 4xe PHEV issues - Hybrid system faults, charging problems
  • Water leaks - Freedom top, doors leak

JK/JKU (2007-2018)

  • Death wobble - Front suspension shimmy
  • 3.8L oil consumption - 2007-2011 engine burns oil
  • TIPM failure - Totally Integrated Power Module issues
  • Exhaust manifold crack - 3.6L manifold cracks
  • Water leaks - Hardtop, soft top, doors

TJ/LJ (1997-2006)

  • Death wobble - Track bar, ball joints, tie rod
  • 0331 head crack - 2000-2001 4.0L cylinder head
  • Frame rust - Frame rails, body mounts
  • NSG370 trans issues - 6-speed grinding
  • Exhaust manifold crack - 4.0L manifold cracks

YJ (1987-1995)

  • Frame rust - Frame rails, spring mounts
  • CPS failure - Crankshaft position sensor (4.0L)
  • Fuel injection issues - TBI/MPI injector problems
  • Renix ECU issues - 1987-1990 computer problems
  • Ball joint wear - Front axle wear
