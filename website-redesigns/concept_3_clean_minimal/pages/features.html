Features

Professional tools.
In your pocket.

Every feature in ODB+ is designed with one goal: helping you understand your vehicle without needing a $5,000 scan tool or an engineering degree.

AI-Powered Diagnostics

ODB+ includes a conversational AI assistant that has access to your vehicle's live sensor data. Ask it anything: "Why is my check engine light on?", "Is this coolant temperature normal?", or "What does P0171 mean for my specific car?"

The AI cross-references trouble codes with your live data to provide contextual, vehicle-specific guidance. It can suggest repair procedures, estimate severity, and help you decide whether to drive to the shop or call a tow truck.

  • Context-aware responses using live data
  • Plain-language repair explanations
  • Severity assessment for each code
  • Step-by-step troubleshooting guides

Live Data Monitoring

Watch your vehicle's vital signs in real time. ODB+ displays over 120 standard OBD-II parameters with clean, readable gauges that update multiple times per second. Monitor everything from engine RPM and coolant temperature to oxygen sensor voltages and catalytic converter efficiency.

Group sensors by system, create custom dashboards, and set alerts when values leave normal ranges. The app highlights anomalies automatically so you can focus on what matters.

  • 120+ supported parameters
  • Real-time updates at up to 10Hz
  • Custom dashboard layouts
  • Automatic anomaly detection

18,000+ Error Code Database

Every standard OBD-II generic code, plus thousands of manufacturer-specific codes for over 50 makes. Each entry includes a plain-language description, common causes ranked by likelihood, and community-verified repair procedures with difficulty ratings.

Search by code number, symptom description, or affected system. Bookmark frequently referenced codes and share them with your mechanic or community forums.

  • Generic and manufacturer-specific codes
  • Ranked common causes
  • Community repair procedures
  • Full-text search

Terminal Mode

For users who want raw control, Terminal Mode provides direct access to OBD-II PIDs and ELM327 AT commands. Send any command, see the raw response, and inspect headers and byte-level data.

Includes full command history with search, autocomplete for common commands, and syntax-colored output that distinguishes headers, data bytes, and status messages.

  • Raw OBD-II and AT command access
  • Syntax-highlighted responses
  • Command history and autocomplete
  • Hex/ASCII response parsing

Session Logger and Replay

Record entire diagnostic sessions -- every command, every response, every sensor reading -- with precise timestamps. Review sessions later to track intermittent issues or compare readings before and after repairs.

Share session files with mechanics, forums, or other ODB+ users. Replay mode lets you scrub through time to see exactly what your vehicle was doing at any moment.

  • Full session recording with timestamps
  • Timeline scrubbing and replay
  • Export and share session files
  • Before/after comparison
