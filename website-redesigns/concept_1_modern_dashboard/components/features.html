Features

Everything You Need to Diagnose

Professional-grade vehicle diagnostics with an intuitive interface. From live data to AI analysis, ODB+ covers every aspect of modern OBD-II diagnostics.

AI Diagnostics

Chat with an AI assistant that understands your vehicle's live data. Get instant repair recommendations, root cause analysis, and step-by-step guidance.

Live Monitoring

Real-time streaming of RPM, temperatures, O2 sensors, MAF, MAP, fuel trims, and dozens more parameters with configurable gauge layouts.

Error Code Database

Access 18,000+ DTCs with full descriptions, probable causes, repair guides, and difficulty ratings. Covers all manufacturers and protocols.

Session Logger

Record diagnostic sessions with full PID data for later replay and analysis. Export logs as CSV or share with your mechanic.

Terminal Mode

Direct access to the OBD-II command interface. Send raw AT and PID commands, view hex responses, and debug adapter communication.

Guided Tests

Step-by-step diagnostic procedures like RPM tests, diagnostic HUD overlays, and health check routines with pass/fail scoring.
