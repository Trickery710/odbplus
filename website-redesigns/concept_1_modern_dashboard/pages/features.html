10 Core Features

Built for Real Diagnostics

Every feature in ODB+ was designed with real-world diagnostic workflows in mind -- from quick code reads to deep data analysis.

AI-Powered

AI Diagnostics

Chat with an AI assistant that has full context of your vehicle's live sensor data, stored trouble codes, and freeze frame information. Ask questions in natural language and get actionable repair advice.

  • Natural language questions about your vehicle's health
  • Root cause analysis with probability scoring
  • Step-by-step repair instructions with difficulty ratings
  • Estimated repair costs and part numbers
Real-Time

Live Data Monitoring

Stream real-time sensor data from your vehicle's ECU with sub-100ms latency. Monitor RPM, coolant temperature, O2 voltage, mass airflow, manifold pressure, fuel trims, and dozens more parameters simultaneously.

  • 40+ standard OBD-II PIDs with automatic discovery
  • Configurable gauge, list, and graph views
  • Min/max/average tracking per session
Comprehensive

18,000+ Error Codes

The most complete DTC database available on mobile. Every generic P0xxx code, plus manufacturer-specific codes for all major brands. Each entry includes a plain-language description, probable causes ranked by likelihood, repair difficulty, and estimated cost.

  • Generic (P0xxx), Enhanced (P1xxx), and Body/Chassis codes
  • Read and clear codes with one tap
  • Freeze frame data for each stored code
Record & Replay

Session Logger & Replay

Record complete diagnostic sessions with timestamped PID data for later analysis. Perfect for tracking intermittent issues, documenting test drives, or sharing data with a professional mechanic.

  • Export as CSV, JSON, or shareable link
  • Replay sessions with time scrubbing
  • Annotate sessions with notes and bookmarks
Advanced

Terminal Mode

Full raw command access to the OBD-II adapter. Send AT configuration commands, custom PID requests, or protocol-specific queries. View raw hex responses with syntax highlighting. Essential for developers and advanced diagnosticians.

  • Command history and favorites
  • Hex response highlighting and byte parsing
  • AT command reference built in
Interactive

Guided Tests

Follow structured diagnostic procedures with real-time pass/fail feedback. RPM steady-state tests, coolant warm-up curves, O2 sensor response checks, and more. The diagnostic HUD provides an at-a-glance overlay of critical parameters.

  • Step-by-step instructions with live data validation
  • Diagnostic HUD overlay mode
  • Exportable test reports

And More

ODB+ goes beyond basic diagnostics with community features, reference materials, and hardware tools.

Vehicle Forums

Browse and post in manufacturer-specific forums covering 50+ brands. Get answers from fellow owners and experienced mechanics.

Sensor Reference

Comprehensive sensor library with normal operating ranges, testing procedures, wiring diagrams, and replacement part numbers.

Parts Catalog

Search for replacement parts by trouble code or component. Compare prices across retailers and find the right part for your exact vehicle.

Adapter Compatibility

Verified compatibility with 100+ ELM327 adapters. Automatic detection of adapter family, firmware version, and supported capabilities.
